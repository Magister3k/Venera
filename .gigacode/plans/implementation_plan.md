# План реализации проекта Venera

## Общая информация
- **Проект**: Venera - Система сбора идентификаторов в потоке пакетных данных
- **Платформа**: Windows x64
- **Язык**: Go
- **Версия**: 1.0.0
- **Статус**: Планирование

### 1.3 Файлы конфигурации и настроек

#### 1.3.1 config.toml (полный)
```toml
[Generic]
mode = "tray"  # tray или service
auto_start_processes = true
max_processes = 20
web_server_port = 8080
disk_warning_percent = 80
ram_warning_percent = 85
max_queue_size = 1000
processing_interval = 30

[Paths]
podman_path = "C:\\Program Files\\Podman\\podman.exe"
tshark_path = "C:\\Program Files\\Wireshark\\tshark.exe"
filter_file = "settings\\generic.flt"
control_file = "settings\\generic.ctr"
alerts_file = "settings\\generic.alr"
dragonfly_image = "docker.io/dragonflydb/dragonfly:latest"
dragonfly_backup_path = "C:\\Venera\\dragonfly_backup"

[DragonflyDB]
host = "localhost"
port = 6379
password = ""
database = 0
batch_size = 1000
timeout = 30

[PostgreSQL]
host = "localhost"
port = 5432
database = "Venera"
user = "venera"
password = "venera"
max_connections = 20

[Logging]
level = "info"
directory = "Logs"
max_age_days = 30
max_files = 10
compress_old_logs = true
```

#### 1.3.2 processes.toml (шаблон)
```toml
[process_1]
id = "process_1"
type = "network"
name = "Network Source 1"
ip = "192.168.1.100"
port = 5000

[process_2]
id = "process_2"
type = "folder"
name = "Folder Source 1"
path = "C:\\Data\\Input"
scan_subfolders = true
monitor_new_files = true

[process_3]
id = "process_3"
type = "file"
name = "File Source 1"
path = "C:\\Data\\input.pcap"
```

#### 1.3.3 generic.flt (шаблон)
```
+source|Источник данных
-192.168.1.1|Запрещенный IP
-10.0.0.1|Запрещенный внутренний IP
+key1|Ключ 1
-value1|Запрещенное значение
+key2|Ключ 2
```

#### 1.3.4 generic.ctr (шаблон CSV)
```
key1,value1
key2,value2
key3,value3
```

#### 1.3.5 generic.alr (шаблон CEL)
```
alert_id = "alert_1"
name = "High CPU Usage"
condition = "cpu_percent > 90"
action = "notify"
severity = "high"
```
